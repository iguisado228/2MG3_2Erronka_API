# Ongi Etorri Gure API Dokumentaziora!

Gure APIa zure aplikazioak gure zerbitzuekin integratzeko diseinatuta dago. Hemen, behar duzun informazio guztia aurkituko duzu, hasi eta amaitu arte.

## Zer Aurkituko Duzu Hemen?

*   **Kontrolatzaileak:** Gure APIaren amaierako puntuak (endpoints) esploratu. Ikasi zein ruta dauden, zertarako balio duten eta zer erantzun itzuliko duten.
    *   [API Kontrolatzaileak eta DTOak](api/toc.html)

*   **DTOak (Data Transfer Objects):** APIaren eta zure aplikazioaren artean trukatzen diren datu-egiturak ulertu. Zein datu mota diren eta noiz erabili behar diren jakin.
    *   [API Kontrolatzaileak eta DTOak](api/toc.html)

*   **Hasteko Gida:** APIa erabiltzen hasteko pauso errazak. Autentifikaziotik hasi eta lehen deiak egiteko argibideak.
    *   [Hasteko Gida](getting-started.md)

*   **Sarrera:** Gure APIaren ikuspegi orokorra eta dokumentazio hau nola erabili.
    *   [Sarrera](introduction.md)

## Nola Erabili Dokumentazio Hau?

Ezkerreko nabigazio-barra erabil dezakezu atal desberdinetan zehar mugitzeko. Bilatzailea ere erabil dezakezu informazio zehatza azkar aurkitzeko.

Gure helburua da APIaren integrazioa ahalik eta errazena izatea. Edozein zalantza edo iradokizun izanez gero, ez izan zalantzarik gurekin harremanetan jartzeko.
